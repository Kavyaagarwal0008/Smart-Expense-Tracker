# Smart Expense Tracker

A responsive expense tracking web application built using HTML, CSS, and JavaScript.

## Features

- Add Expense
- Delete Expense
- Search Expense
- Category Filter
- Monthly Summary
- Local Storage Support
- Responsive Dashboard UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## Future Enhancements

- Edit Expense
- Pie Charts
- Budget Alerts
- Export PDF
- Dark Mode
